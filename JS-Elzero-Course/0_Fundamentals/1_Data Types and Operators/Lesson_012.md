## Rules for Naming Variables  :

1. NO Spaces.
2. Can't start with a number.
3. Can start and use _ and $.
4. Can't use any special characters.
5. Names are case sensitive.
6. Can't use Reserved Words.

##  We use Camel Case Way to Name Variables :

1. Start with a lowercase letter.
2. Capitalize the first letter of each subsequent word.
