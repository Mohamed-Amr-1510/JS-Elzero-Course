### Do While Loop Syntax :

```js
initialize

do{
//Block of code
Update

}while(Condition)
```

### Difference between Do...While and ( While , For) :

**Do...While :**
 1. Execute the code once.
 2. Check the condition.

**While , For :**
 1. Check the condition.
 2. Execute the code.
