# How to make Comments :

*We can make comment and uncomment by : Ctrl + /*
## 1. Single Line Comment : 
  To do it we use : //
```js
// Single Line Comment
```
We also can use :  /* \*/
```js
/* Single Line Comment */
```
## 2. Multiple Line Comment : 
  To do it we use :  /* \*/
```js
/*
  1
  2
  3
  4
  5
*/
```
